在高中阶段，我们很喜欢数列，但是我们没有探究数列最终的走向，要研究这个，我们就要研究数列的极限

定理0：实数的完备性

它是说，对于实数集R的一个子集A和另外一个数x，若x大于A的所有元素，则x是A的上界，A的最小的上界是A的上确界，完备性是说，如果A有上界，那么A必须有上确界，下界和下确界是类似定义的

定理1：数列的极限

假设我有一个数列{a[n]}，对任意的ε>0，存在N，s.t. 当n>N时，有ε>|a[n]-a|（a是任意实数），那么我说，{a[n]}收敛到a，记作lim a[n]=a，否则我说{a[n]}发散

从几何学的观点来看，对于包含a的区间(a-ε,a+ε)，数列{a[n]}只有有限个点落在区间外面

例题0：求证：调和级数是发散的

证明：调和级数，也就是a[n]=Σ1/i （i∈[1,n]）

我们知道，1/i > ln(1+1/i)（这个的证明我们后面再给出，但这是高中数学的常见结论）

假设调和级数收敛，且收敛到a，那么对于任意的ε>0，存在N，s.t. n>N均有ε>|a[n]-a|

但是，a[n] > Σln(1+1/i) = ln(n)

而，我取一个n，使得n > exp(a+1)，会有|a[n]-a|>1，这和极限的定义矛盾

所以调和级数是发散的。Q.E.D.

定理2：数列不能有很多个不一致的极限

证明：假设可以，设{a[n]}的极限是a和b，lim a[n] = a = b

那么，对于任意的ε，分别存在N[1]和N[2]，s.t. n>N[1]时，ε/2>|a[n]-a|，n>N[2]时，ε/2>|a[n]-b|

那么，n>N[1]+N[2]时，二者都要成立，我们把他们加起来有ε>|a[n]-a|+|b-a[n]|>=|b-a|

对于ε足够小，这都要成立，那么只能b-a=0。Q.E.D.

定理3：改变数列的有限个项，数列的极限是不会改变的。

证明：显然。设改变的最后一项的下标是i，将N修改为N+i，会发现根本没变。Q.E.D.

定理4：去掉数列的有限个项，数列的极限不变

证明：请你参考定理3

定理5：收敛数列必须有界

如果{a[n]}收敛，那么存在M，使得M>=|a[n]|

证明：假设a[n]收敛到a，则对于任意的ε>0，存在N，使得n>N时，有ε>|a[n]-a|

那么取ε=1，则1>|a[n]-a|

那么a+1>a[n]

那么计M=max(a[1],a[2],...,a[N],a+1)

max里面的项是有限的，一个有限的集合必定有确定的最大值，且显然M>=a[n]。Q.E.D.

定理6：极限的四则运算

如果lim a[n] = a，lim b[n] = b，那么lim(a[n]+b[n]) = a+b

lim(a[n]-b[n]) = a-b

lim(a[n]*b[n]) = a*b

lim(a[n]/b[n]) = a/b（如果b!=0）

证明：那么，分别存在N[1]和N[2]，s.t. 对于任意的ε，当n>N[1]时，ε/2>|a[n]-a|，n>N[2]时，ε/2>|b[n]-b|

那么，当n>N[1]+N[2]时，ε>|a[n]-a|+|b[n]-b|>=|a[n]+b[n]-a-b|

也就是lim (a[n]+b[n]) = a+b

ε>|a[n]-a|+|b[n]-b|=|a[n]-a|+|b-b[n]|>=|(a[n]-a)+(b-b[n])|=|a[n]-b[n]-(a-b)|

也就是lim (a[n]-b[n]) = a-b

|a[n]b[n]-ab|=|a[n]b[n]-ab+ab[n]-ab[n]|=|(a[n]-a)b[n]+a(b[n]-b)|

定理5告诉我们，收敛数列一定有界，计一个很大的M，可以使得M>=|a[n]|,|b[n]|

同时，存在N[1]和N[2]，s.t. 当n>N[1]时，ε/(2M)>|a[n]-a|，当n>N[2]时，ε/(2M)>|b[n]-b|

取N=N[1]+N[2]，当n>N时

ε=ε/(2M)*M+ε/(2M)*M>|a[n]-a||b[n]|+|a||b[n]-b|=|(a[n]-a)b[n]|+|a(b[n]-b)|>=|(a[n]-a)b[n]+a(b[n]-b)|=|a[n]b[n]-ab|

也就是lim a[n]b[n] = a*b

|1/b[n]-1/b|=|(b[n]-b)/(b*b[n])|=|b[n]-b|/|b*b[n]|

显然，对于任意的ε>0，存在N，使得当n>N时，|b||M|ε>|b[n]-b|

那么，ε=|b[n]-b|/|b*M|>|b[n]-b|/|b*b[n]|=|1/b[n]-1/b|

所以，lim 1/b[n] = 1/b

所以，lim a[n]/b[n] = a/b。Q.E.D.

例题1：求证：sin(n)是发散的

证明：假设sin(n)是收敛的，lim sin(n) = a

显然，lim sin(n+2) = a

lim (sin(n+2)-sin(n)) = 0

和差化积，lim (2cos(n+1)sin1)=0

也就是lim cos(n+1) = 0

即lim cos(n) = 0

lim (cos(n+2)-cosn) = 0

再次和差化积，有lim (-2sin1sin(n+1)) = 0

也就是lim sin(n+1) = 0

lim sin(n) = 0

利用极限运算的性质，lim 1=lim (pow(sin(n),2)+pow(cos(n),2)) = 0

这是很荒谬的事情，所以假设不成立，sin(n)是发散的

定理7：夹逼定理

如果b[n]>=a[n]>=c[n]恒成立，且lim b[n] = lim c[n] = a，那么lim a[n] = a

证明：那么，对于任意的ε>0，存在N[1]和N[2]，s.t. n>N[1]时，有ε>|b[n]-a|，n>N[2]时，有ε>|c[n]-a|

那么，取N=N[1]+N[2]，n>N时，ε>b[n]-a>=a[n]-a>=c[n]-a>-ε，也就是ε>|a[n]-a|，所以lim a[n]=a。Q.E.D.

例题2：求证：对于任意的a>0，lim pow(a,1/n)=1

证明：a>=1时，pow(a,1/n)>=1

由均值不等式，(a+n-1)/n>=pow(a,1/n)

显然，lim (a+n-1)/n = lim 1 + lim (a-1)/n = 1

lim 1 = 1

夹逼定理得到lim pow(a,1/n) = 1

1>a>0时，1>pow(a,1/n)>n/(1/a+(n-1))=1/(1/(a*n)+1-1/n)，这是因为调和平均数不会超过几何平均数

lim 1/(1/(a*n)+1-1/n) = 1

所以，lim pow(a,1/n)仍然是1。Q.E.D.

例题3：求证：pow(n,1/n) = 1

证明：显然，pow(n,1/n) = pow(sqrt(n)*sqrt(n),1/n)

接下来，由于(2/sqrt(n)+1-2/n)=(2sqrt(n)+n-2)/n>pow(n)>1

左右极限都是1，故lim pow(n,1/n) = 1。Q.E.D.
